# Shopp - Modern E-commerce Website

🛍️ **เว็บไซต์อีคอมเมิร์ซที่ทันสมัยพร้อมเอฟเฟกต์ล้ำยุค**

## ✨ Features

- 🎨 **Modern UI/UX Design** - ออกแบบด้วย CSS Grid, Flexbox และ Advanced Animations
- 📱 **Responsive Design** - รองรับทุกอุปกรณ์ (Desktop, Tablet, Mobile)
- 🌈 **Advanced Visual Effects** - Glassmorphism, Gradient Backgrounds, Smooth Transitions
- 🛒 **Shopping Cart System** - ระบบตะกร้าสินค้าที่ทันสมัย
- 👤 **User Authentication** - ระบบสมาชิกและการจัดการผู้ใช้
- 🔍 **Product Search & Filter** - ค้นหาและกรองสินค้าได้อย่างรวดเร็ว
- 💳 **Payment Integration** - รองรับการชำระเงินหลายรูปแบบ
- 📊 **Admin Dashboard** - ระบบจัดการสำหรับผู้ดูแล

## 🚀 Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+), Bootstrap 5
- **Backend**: PHP 8+, MySQL
- **Libraries**: 
  - Font Awesome (Icons)
  - AOS (Animate On Scroll)
  - SweetAlert2 (Beautiful Alerts)
  - Google Fonts (Typography)

## 📦 Installation

1. **Clone Repository**
   ```bash
   git clone https://github.com/GitBababoo/Shopp.git
   cd Shopp
   ```

2. **Setup Database**
   - Import `database.sql` to your MySQL server
   - Update database configuration in `config/database.php`

3. **Configure Web Server**
   - Point your web server to the project directory
   - Ensure PHP 8+ and MySQL are installed

4. **Access Website**
   - Open your browser and navigate to your local server
   - Default admin: `admin@shopp.com` / `admin123`
